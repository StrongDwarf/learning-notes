## 1,nginx简介

在部署了nginx的服务器中,其网络结构一般为: internet -> 防火墙 -> F5 -> nginx服务器 -> app服务器 -> DB。

###### nginx主要优点:

* 1, 高并发连接 : 官方测试能够支撑5万并发连接, 在实际生产环境中跑到2~3万并发连接数
* 2, 内存消耗小 : 在3万并发连接下, 开启的10个nginx进程才消耗150M内存
