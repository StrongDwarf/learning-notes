## 1,linux命令概述

### 1.1 linux操作系统启动流程梳理

linux系统启动流程可以简单总结为以下几步:

**开机BIOS自检** -> **可启动设备检查** -> **执行引导程序启动设备** -> **加载内核** -> **启动初始化进程** -> **启动内核模块** -> **启动mingetty,进入系统登录界面**

* 开机BIOS自检 : 基本输入输出系统完成, 硬件检查, 可启动设备检查, 最终完成加载硬盘。
* 可启动设备检查 : 读取可启动设备, 让主引导记录进行可启动设备引导。
